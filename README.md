# simulador_heuristica

Criar Virtual Enviroment:

python3 -m venv venv 

Executar Virtual Enviroment:

Linux: source venv/bin/activate  

Windows: venv/Scripts/activate

Instalar dependencias:

pip install -r requirements.txt
